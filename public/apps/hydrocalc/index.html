Reference Materials
JIS 給排水衛生設備 — pipe data loaded from external reference file
Pipe reference data is loaded from an external pipe chart reference file. This keeps the calculator app light and the reference data independently editable. Load the file once per session — all sizing calculations will use it automatically.
Fixture Library
